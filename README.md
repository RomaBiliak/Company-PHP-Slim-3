# Company-PHP-Slim-3
## Встановлення:
* git clone https://github.com/RomaBiliak/Company-PHP-Slim-3.git .
* composer update
* Створити базу Mysql, дамп бази знаходиться в каталозі sql/
* Настроїти підключення до бази bootstrap/app.php
  ```
  'host' => 'localhost',
  'database' => 'database',
  'username' => 'root',
  'password' => '123456',
  ```
